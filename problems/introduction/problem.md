# 入門

首先，先為這個 Workshop 建立一個目錄。

在終端機中執行下面的命令，或是你可以用任何其他方式建立目錄：

`mkdir javascripting`

切換到這個 `javascripting` 目錄之下：

`cd javascripting`

然後，建立一個檔案，並命名為 `introduction.js`。並在這檔案中寫入下列程式碼內容：

```js
console.log('hello');
```
 
最後，你可以用下列命令檢查你的程式執行結果是否正確：

`javascripting verify introduction.js`
